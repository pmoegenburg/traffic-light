
# traffic-light

This project simulates a traffic light state machine and includes tests for verifying the core state transition logic.

## Usage

**Build and Run the Simulation**:
   ```bash
   make
   ./traffic_light
   ```

**Build and Run Tests**:
   ```bash
   make test
   ./test_traffic_light
   ```

**Clean Up**:
   ```bash
   make clean
   ```
